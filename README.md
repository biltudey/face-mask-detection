## Run Locally

Install the below libary

```bash
    pip install tensorflow

```
```bash
    pip install opencv-python

```

Clone the project

```bash
  gh repo clone biltudey/face-mask-detection
```

Go to Your  project directory

```bash
  cd your-project
```

Start the App

```bash
  python main.py
```
